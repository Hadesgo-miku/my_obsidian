### **🩺 高级组合特征清单（基于你的特征）**
| **新特征名**                     | **组合公式**               | **生理解释与意义**              |
| ---------------------------- | ---------------------- | ------------------------ |
| **FEV1_to_FVC_ratio**        | FEV1 / FVC             | 已有，气流受限程度，低说明阻塞          |
| **FEV3_minus_FEV1_over_FVC** | (FEV3 - FEV1) / FVC    | 已有，反映呼气后段的排气功能           |
| **PEF_over_MMEF**            | PEF / MMEF             | 呼气初期与中期流速的比例，反映大气道与小气道匹配 |
| **MEF50_over_MEF25**         | MEF50 / MEF25          | 中段与末段呼气速度比，小气道受限早期信号     |
| **PEF_variability**          | abs(PEF% - 100)        | 呼气峰值偏离正常水平的幅度，气道功能异常提示   |
| **FVC_vs_predicted**         | FVC% / 100             | 用力肺活量达到预测标准的比例，整体肺功能评估   |
| **FEV1_vs_predicted**        | FEV1% / 100            | 第一秒呼气容积达到预测标准的比例，气流量评价   |
| **WBC_times_EOS**            | WBC × EOS              | 炎症状态综合指标，提示气道炎症程度        |
| **BMI_times_Age**            | BMI × Age              | 体型和年龄交互特征，影响呼吸道反应性       |
| **MMEF_ratio**               | MMEF / FEV1            | 小气道流速与大气道容积能力比，敏感检测小气道阻塞 |
| **MEF25_slope**              | (MEF50 - MEF25) / 25   | t呼气流量下降速率，反映小气道开放性       |
| **Volume_deficit**           | (FVC - FEV1) / FVC     | 失去的初期呼气容量比例，气流受限早期指标     |
| **Airway_Resistance_Index**  | (1 - (FEV1/FVC)) × 100 | 气道阻力估算值（单位化处理）           |
| **EOS_relative**             | EOS / WBC              | 嗜酸性粒细胞相对比例，提示过敏性炎症       |
| **PEF_to_Height**            | PEF / Height           | 呼气峰流速与身高比例，标准化呼气能力       |
